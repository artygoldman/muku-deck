6. QUALITY REQUIREMENTS / 品質要件

6.1 Transit Testing / 輸送試験

Test / 試験 Standard / 規格 Requirement / 要件
Drop Test / 落下試験 ISTA 3A 76 cm drop height, 10 drops on all faces, edges, and corners. No product damage, no box failure. / 76cm高さから10回落下（全面・辺・角）。製品損傷・箱破損なきこと。
Edge Crush Test (ECT) / エッジクラッシュ試験 ISO 3037 Minimum 5.5 kN/m / 最低5.5 kN/m
Cobb Test (Water Absorption) / コッブ吸水試験 ISO 535 Maximum 35 g/m² (30 min exposure) / 最大35 g/m²（30分暴露）

6.2 Inspection Levels (AQL) / 検査水準（AQL）

Defect Level / 欠陥レベル AQL Value / AQL値 Description / 説明
Critical / 致命的 0 (zero tolerance) Safety hazard, regulatory non-compliance, wrong product / 安全上の危険、法規制違反、誤製品
Major / 重大 2.5 Structural damage, print defects visible to customer, missing components / 構造的損傷、顧客可視の印刷不良、部品欠品
Minor / 軽微 4.0 Slight color variation, minor scuff on kraft, small alignment offset / 軽微な色差、クラフト面の小さな擦れ、わずかな位置ズレ
All tests must be conducted by an accredited third-party lab and documented with photographic evidence. Test reports are required before mass production approval.
全試験は認定第三者機関にて実施し、写真付きで記録すること。量産承認前に試験報告書の提出が必要です。

8. PACKING PROCESS / 梱包プロセス

Fig. 8-1: 9-Step Packing Process / 9ステップ梱包プロセス
1
Fold Box / 箱を折る
Erect crash-lock bottom box. Verify corners are square and lock engages fully. / クラッシュロックボトムを立ち上げ。角の直角とロック完全係合を確認。
2
Insert First Bracket / 第1ブラケット挿入
Place left U-bracket insert at the bottom-left end of the box. / 左側Uブラケットを箱の左端底部に設置。
3
Wrap Product / 製品を包む
Wrap the bamboo tray in kraft paper. Fold ends neatly, no tape on product. / 竹トレーをクラフト紙で包む。端部を丁寧に折り、製品にテープ不可。
4
Place Product / 製品を配置
Slide wrapped tray into box. Telescopic arms must be fully retracted (700 mm). Mini-tray and clip placed on top. / 包装済みトレーを箱に挿入。伸縮アームは完全収縮（700mm）。ミニトレー・クリップは上に配置。
5
Insert Second Bracket / 第2ブラケット挿入
Place right U-bracket insert at the right end, firmly against the product. / 右側Uブラケットを右端に設置、製品にしっかり当接。
6
Place Card on Top / カードを載せる
Lay QR care card face-up on top of the wrapped product. Visible upon opening. / QRお手入れカードを表面上向きで包装済み製品上に配置。開封時に見えるように。
7
Close Lid / 蓋を閉じる
Close top flaps. Seal with single strip of kraft tape along the long center seam. / 天面フラップを閉じる。長辺中央シームにクラフトテープ1本で封緘。
8
Apply FNSKU / FNSKU貼付
Apply FNSKU label on right short side. Ensure no wrinkles, full adhesion, correct orientation. / FNSKUラベルを右短辺に貼付。シワなし、完全接着、向き正確。
9
QC Check / 品質検査
Verify: box integrity, label scan, weight check (1.95 kg ±50g), visual inspection. Pass = place in master carton. / 箱の完全性、ラベルスキャン、重量確認（1.95kg ±50g）、目視検査。合格＝マスターカートンへ。
